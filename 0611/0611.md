## Primitive data type
- Primitive data type 이란 python에서 제공하는 / 정의한 기본 데이터 타입을 말한다.

    |  Type |   설명   |   Example   |
    |:-----:|:--------:|:-----------:|
    |  int  |   정수   |  -10, 0, 32 |
    | float |   소수   |  8.231, 1.0 |
    |  str  | 문자(열) | 'Hi', "123" |
    |  bool |  참/거짓 | True, False |

- int / float 의 연산
    - 사칙연산: +, -, \*, /,
    - \*\*: 승. 제곱
    - %: 나머지
    - //: 몫

- 형 변환
    - int()
    - float()
    - str()
    - bool()
- 기본 연산
    - 숫자
        - + - * / : 사칙연산
        - ** : 승
        - % : 나머지
        - // : 몫
    - 문자
        - + : concatenate
        - *
        - indexing: index번째의 문자를 가져오는 행위
            - index: 번째 = 2
            - 음수도 가능
            - 'abc'[2]
        - slicing
            - [from index 이상: to index 미만]
            - 'abcdef'[0: 3]
            - 'abcdef'[0:]
            - 'abcdef'[:3]
            - 'bac'[:]
            - 'abcdef'[-3: -1]
        - len('abc')
        - membership
            - in
                - 'a' in 'abc'
                - 'abc' in 'abc'
                - 'ba' in 'abc'
            - not in
                - 
    - Boolean
        - not
            - not True
            - not False
            - not ('a' in 'abc')
            - not 'a' in 'abc'
                - 적어드릴게요
            - 'a' not in 'abc'
        - or: 합집합 +
            - True or True == True  
            - True or False == True
            - False or True == True
            - False or False == False
        - and: 교집합 * 
            - True and True == True 
            - True and False == False
            - False and True == False
            - False and False == False

### Loop 문 제어
- break
    - 중도 중단
    - break를 만나면 그 loop는 즉시 중단된다.
    - 예)
        ```python
        lst = [0, 1, 2, 3, 1, 2, 3, 4]

        for e in lst:
            if e > 2:
                break
            print(e)
        ```
        출력 결과
        ```
        0
        1
        2
        ```
- continue
    - 스킵
    - continue를 만나면 즉시 다음 iteration으로 넘어간다
    - 예)
        ```python
        lst = [0, 1, 2, 3, 1, 2, 3, 4]

        for e in lst:
            if e > 2:
                continue
            print(e)
        ```
        출력 결과
        ```
        0
        1
        2
        1
        2
        ```

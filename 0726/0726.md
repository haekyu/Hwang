|              |       c++       |        python        |
|:------------:|:---------------:|:--------------------:|

- Hello World!
    - `hello.cpp` 작성
        ```c
        #include <iostream>
        using namespace std;

        int main(){
            cout << "Hello World!\n";
            return 0;
        }
        ```
    - 컴파일 (터미널에서)
        ```
        g++ hello.cpp -o hello.out
        ```
        - g++ 컴파일러로 
        - hello.cpp 코드를 컴파일하여 
        - -o 아웃풋을 내어 놓는다.
        - hello.out 이 아웃풋 이름이다.
    - 실행 (터미널에서)
        - `./hello.out

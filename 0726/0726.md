## 기본 연산
- 숫자 연산
    - 사칙 연산: +, -, \*, /
    - 나머지: %
- 문자(char, string) 연산
    - +
        - concatenate, 붙이기
        - 예)
            ```c
            // 변수 정의
            string s = "ABC";
            char c = 'd';
            
            // concatenate
            string long_string = s + c;
            
            // 결과 출력
            cout << long_string << "\n";
            ```
            출력 결과
            ```
            ABCd
            ```
    - `size()`, `length()`
        - string 길이
        - 예)
            ```c
            // 변수 정의
            string s = "ABC";
            char c = 'd';
            
            // concatenate
            string long_string = s + c;
            
            // long_string의 길이 출력
            cout << long_string.size() << "\n";
            ```
            출력 결과
            ```
            4
            ```
    - string의 인덱싱
        - 특정 인덱스에 위치한 char을 얻음
        - `at()` 함수 사용
        - 파이썬과 다르게 음수 인덱스 없음
        - 예)
            ```c
            // string 정의하기
            string base = "hello world!";

            // string 인덱싱. 0번째 문자 가져오기
            char fst_char = base.at(0);

            // 인덱싱 결과 확인하기
            cout << fst_char << "\n";
            ```
            출력 결과
            ```
            h
            ```
    - string에 있는 특정 문자 검색
        - 전체 문자열 중에서 원하는 특정 문자열이 어디에 위치해 있는지, 그 인덱스를 얻음
        - `find()` 함수 사용
        - 예)
            ```c
            // string 정의하기
            string base = "hello world!";

            // 문자열 검색
            int idx_of_world = base.find("world!");

            // 문자열 검색 결과 확인하기
            cout << idx_of_world << "\n";
            ```
            출력 결과
            ```
            6
            ```
    - 문자열 비교
        - 두 string을 비교한다.
        - `compare()`함수 사용.
            - 두 string이 얼마나 차이가 나는지 숫자로 리턴해줌.
            예) string a와 b를 비교
                ```c
                string a = "ABC";
                string b = "AB";
                int cmp = a.compare(b);
                ```
            - 결과 0
                - 차이가 0
                - 두 string이 같다.
            - 결과 > 0
                - a > b
                - a가 b보다 사전 순으로 뒤에 있다.
            - 결과 < 0
                - a < b
                - a가 b보다 사전 순으로 먼저 있다.
